Unlike sharp or gnawed edges, Ember uses smooth cubic-bezier waves. Each style creates a different mood — from barely-there ripples to bold rolling swells.

Wave 01 — Gentle Single
A single slow arc. Barely noticeable, creates elegant flow. Best for light transitions.

Usage Notes
When to use steam
Hero sections above coffee product imagery · Product card icon areas · Ambient texture on warm-colored CTAs · Loading states (ambient)
Performance tips
Use will-change: transform, opacity on wisps. Keep to max 3 simultaneous wisp animations per viewport. Use filter: blur() only on small elements.
