03 — Typography

Type Scale

A fluid scale using clamp() for responsive sizing. Display and H1–H3 scale with the viewport; H4–body stay fixed.

Token Size Family Preview
--text-display clamp(44px→64px) Playfair Display Jessna Rodriguez
--text-h1 clamp(32px→48px) Playfair Display Full-Stack Developer
--text-h2 clamp(26px→36px) Libre Baskerville Selected Projects
--text-h3 clamp(22px→28px) Libre Baskerville About Me
--text-h4 20px / 1.25rem Libre Baskerville Coffee Tracker App
--text-lg 18px / 1.125rem Lato Lead paragraph / intro text
--text-base 16px / 1rem Lato Body copy, general UI
--text-sm 14px / 0.875rem Lato Card metadata, labels
--text-xs 12px / 0.75rem Lato / Fira Code Section Label · Tag
